# Tài Liệu Dự Án

Đây là trang mục lục cho toàn bộ tài liệu trong thư mục `docs` của dự án VinhPhat_App.

## Tài liệu chính

- [Giới thiệu tổng quan](intro.md)
  - Mô tả mục tiêu dự án, tính năng chính, kiến trúc kỹ thuật và luồng dữ liệu.
- [Kế hoạch phát triển](PLAN.md)
  - Theo dõi các phase triển khai, task đã hoàn thành và định hướng phát triển.
- [Quy tắc code](RULES.md)
  - Quy ước đặt tên, cấu trúc file và các nguyên tắc code trong dự án.

## Bộ tài liệu QA mobile

- [Checklist Mobile QA](mobile-qa-checklist.md)
  - Danh sách các bước test mobile theo từng page và từng nhóm tương tác.
- [Mobile Bug Log](mobile-bug-log.md)
  - Mẫu ghi nhận lỗi mobile để theo dõi phát hiện, sửa lỗi và retest.

## Luồng làm việc khuyến nghị

1. Đọc [Giới thiệu tổng quan](intro.md) để nắm bức tranh chung của hệ thống.
2. Xem [Kế hoạch phát triển](PLAN.md) nếu cần đối chiếu phạm vi tính năng.
3. Khi test giao diện mobile, bắt đầu từ [Checklist Mobile QA](mobile-qa-checklist.md).
4. Nếu phát hiện lỗi, ghi lại ngay trong [Mobile Bug Log](mobile-bug-log.md).
5. Khi cần chỉnh sửa code, đối chiếu thêm với [Quy tắc code](RULES.md).

## Gợi ý sử dụng nhanh

- Nếu bạn đang onboarding: đọc `intro.md` trước.
- Nếu bạn đang test mobile: mở `mobile-qa-checklist.md` và `mobile-bug-log.md` song song.
- Nếu bạn đang sửa lỗi hoặc thêm tính năng: đọc `RULES.md` và `PLAN.md` trước khi thay đổi lớn.
