# VinhPhat_App

Ứng dụng quản lý kho vải Vĩnh Phát — Nhập liệu và Xuất kho.

## Cấu trúc dự án

- `index.html`: Tệp HTML chính.
- `src/css/`: Stylesheets.
- `src/js/`: JavaScript modules.
- `scripts/`: Google Apps Script.
- `docs/`: Tài liệu.
- `assets/`: Tài nguyên tĩnh.

## Cách chạy

1. Mở `index.html` trong trình duyệt.
2. Cấu hình URL Google Apps Script trong phần cài đặt.

## Phát triển

- Tách CSS và JS từ `index.html` vào các tệp riêng.
- Sử dụng Google Apps Script để đồng bộ với Google Sheets.
